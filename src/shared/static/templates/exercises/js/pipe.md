Реализуй функцию **pipe(...fns)**, которая принимает произвольное количество функций и возвращает новую функцию. Возвращённая функция при вызове последовательно применяет все переданные функции к аргументу, передавая результат каждой следующей функции.

### Требования

1. **Последовательное применение**
   Каждая следующая функция получает результат выполнения предыдущей.

2. **Поддержка асинхронных функций**
   Если любая из переданных функций асинхронная (возвращает **Promise**), она должна корректно обрабатываться.

3. **Возвращаемое значение**
   - Если среди функций есть хотя бы одна асинхронная → возвращается **Promise**
   - Если **все функции синхронные** → всё равно возвращается **Promise**
   - Результат выполнения последней функции разрешается в этом промиса

4. **Пустой pipe()**
   Если вызвать **pipe()** без аргументов, возвращённая функция должна возвращать переданный ей аргумент как есть (обёрнутым в **Promise**).

5. **Порядок выполнения**
   Строго последовательный, не параллельный.

6. **Обработка ошибок**
   Ошибка в любой из функций должна приводить к **rejected Promise**.

```js

// ---- 1 ----
const double = (x) => x * 2;
const addOne = async (x) => x + 1;
const square = (x) => x * x;
const format = async (x) => `Log: ${x}`;

const transform = pipe(double, addOne, square, format);

transform(3).then(console.log); // 'Log: 49'

// ---- 2 ----
const calc = pipe(double, square);
calc(4).then(console.log); // 64

// ---- 3 ----
const identity = pipe();
identity(42).then(console.log); // 42

// ---- 4 ----
const errorFn = x => { throw new Error('Ошибка в функции'); };
const safeFn = x => x + 1;

pipe(errorFn, safeFn)(5)
  .then(result => console.log('Успех:', result))
  .catch(error => console.log('Поймана ошибка:', error.message));

// ---- Impl ----
function pipe() {}
```

---

```js
/** Ответ */

function pipe(...fns) {
  return function (input) {
    return fns.reduce(
      (promise, fn) => promise.then(fn),
      Promise.resolve(input)
    );
  };
}

```

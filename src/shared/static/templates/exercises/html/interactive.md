
```html

<p>Нажмите кнопку, чтобы узнать больше:</p>

<button type="button">
  Перейти к
  <a href="/details">
    Подробностям
  </a>
</button>

```

---

Этот код создает неоднозначность для браузера и вспомогательных технологий:

1. Непредсказуемое поведение парсера: Браузеры пытаются "исправить" этот невалидный HTML. Некоторые могут "выбросить" (hoist) тег *a* из *button*, полностью нарушая предполагаемую структуру.

2. Конфликт событий

3. Проблемы доступности (a11y): Это кошмар для скринридеров. Пользователь услышит что-то вроде: "Кнопка, Перейти к, Ссылка, Подробностям".
